# Files and their purpose
## common-methods.wls
- Contains common methods that all other scripts in this directory use

## all-json.wls
- Contains info common on all tabs of periodic table (simple, properties orbitals, electrons)

## µ.wls
- Conains periodic table info about µ properties, which usually has its own tab

## colors.wls
- Soon to be deprecated
- Contains info for the color values of each periodic table element

## common-methods.wls
- Methods shared with all-json.wls, µ.wls colors.wls, or other Mathemtica files related to this project
